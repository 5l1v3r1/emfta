What is EMFTA?

EMFTA is an EMF implementation of a Fault-Tree Analysis tool. It uses the Sirius framework for the representation. It includes some basic analysis tools.

What is a Cutset?

A cutset is the list of events that are required to raise an error. Each model has one or several cutset, each of them identifies the combination of errors that will trigger an error in the system.

What is AADL and OSATE?

AADL stands for Architecture Analysis and Design Language. This is a modeling language to capture software and system architectures. OSATE is the modeling framework that supports the AADL language. As EMFTA, it is implemented within Eclipse. The AADL has an extension to specify errors and failures. We then add a bridge between EMFTA and OSATE to automatically generate Fault-Tree from AADL models.
